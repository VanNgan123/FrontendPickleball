# Frontend Pickleball

## Cong nghe su dung
- React
- TypeScript
- Vite
- Redux Toolkit
- MUI
- Axios
- React Router

## Cai dat

```bash
npm install
```

Tao file `.env`:

```env
VITE_API_URL=http://localhost:3001
```

## Chay du an

```bash
npm run dev
```

## Build production

```bash
npm run build
```
